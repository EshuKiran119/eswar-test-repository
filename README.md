## ðŸ”¥ Getting Started

To set up and run any project:

   ```

   ```

   ```
   cd <Project-Name>
   ```
3. **Install Dependencies**  
   - For Python projects:
     ```
     pip install -r requirements.txt
     ```

     ```
     npm install
     npx playwright install
     ```
4. **Run Test Cases**  
   - Python API tests:
     ```
     pytest test_suite.py
     ```
   - Playwright tests:
     ```
     npx playwright test
     ```
